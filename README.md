# Konfig (konfig)
Konfig is a developer tools platform that generates SDKs, documentation, demos, and tutorials for REST APIs based on OpenAPI Specifications and Postman Collection Format. It automatically generates and publishes SDKs in all major programming languages including Python, TypeScript, Java, C#, Go, Ruby, PHP, Swift, and Kotlin.

## APIs

### Konfig SDK Generation Platform
Konfig's developer platform for generating, validating, and publishing SDKs, documentation, and interactive tutorials from OpenAPI specifications and Postman Collections. Delivered primarily through the konfig-cli and GitHub Actions integration; Konfig does not currently publish its own public REST API.
